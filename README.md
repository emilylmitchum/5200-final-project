# Project-Repo
Project Group 2, Spring 2026

Authors: Dana Kim, Kylie King, and Emily Mitchum

This is a project for DSAN 5200: Advanced Data Visualization. The goal of the project is to create a compelling data-driven narrative, with an emphasis on visual storytelling. The final product of this repo is a website, which can be viewed at _______ . 

Our selected topic is women's right to vote in the United States, focusing on the recent SAVE Act and its implications. It primarily uses survey data from Pew Research's National Public Opinion Reference Survey and the University of Chicago's NORC AP VoteCast 2024 exit poll survey, along with relevant SAVE Act data from the Center for American Progress. Further details on data sources can be found in the Appendix section of the webpage. 

The 'report' section contains all of the website content. The 'presentation' section contains an informal, short form presentation we completed in class. Respective EDA and chart development can be found in 'dana_eda,' 'kylie_eda,' and 'emily_eda.' 'archive' contains old versions of files that are not part of the product, 'charts' is an incomplete library of charts for the presentation and website. A complete copy of 'charts' can be found in the 'report' section.
